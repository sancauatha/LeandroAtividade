Exércício de HTML com Imagem

Olá, Sou o Uatha, e estou criando primeira página WEB com imagem fora do ambiente de ensino. Neste exercício em específico estou utilizando da base de códigos que o professor nos ensinou em sala de aula, que servem como príncipio para criar uma página WEB. Sendo elas o HTML, onde você vai utilizar no início e fim do seu códigos, que ela que irá fazer a sua página ser uma HTML, o head que basicamente é o cabeçalho do código, o body que você vai botar o corpo do código, onde vai aparecer na página web diretamente e o código de img que vai nos fornecer a imagem para renderizar na página web
